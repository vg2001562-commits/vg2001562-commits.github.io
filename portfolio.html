📚 Как работают алгоритмы

🔄 forEach

Проходит по каждому элементу массива и выполняет указанную операцию. Красным подсвечивается текущий обрабатываемый элемент.

🔍 filter

Создает новый массив с элементами, прошедшими проверку. Зеленые - проходят условие (>4), серые - не проходят.

🎯 search

Линейный поиск элемента. Последовательно сравнивает каждый элемент с искомым (7). Найденный элемент подсвечивается оранжевым.

📊 sort

Пузырьковая сортировка. Сравнивает соседние элементы (оранжевая пара) и меняет их местами, если нужно.

🧮 reduce

Суммирует все элементы массива. Красный - текущий элемент, фиолетовый - аккумулятор, серый - уже обработанные элементы.

🗺️ map

Создает новый массив, применяя функцию к каждому элементу. Удваивает значение каждого числа.

⚡ бинарный поиск

Быстрый поиск в отсортированном массиве. Каждый шаг делит область поиска пополам.
